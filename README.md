> A hybrid app that streams and offers internet using React Native.

### 🏠 [Homepage](https://github.com/fabianmuema/react-native-movie-app)

## Install

```sh
git clone https://github.com/fabianmuema/react-native-movie-app.git
```

## Usage

```sh
yarn run start
```

## Run tests

```sh
yarn run test
```
